# Bouncing Ball EMDR Therapy Tool

https://brilliant-macaron-785655.netlify.app/

## Overview

The Bouncing Ball EMDR Therapy Tool is a WebGL-based application designed to simulate a bouncing ball for Eye Movement Desensitization and Reprocessing (EMDR) therapy. This tool helps users focus and engage through visual and auditory stimuli by providing a continuous and interactive bouncing ball experience.

## Features

- **3D Bouncing Ball:** A ball continuously bounces within the WebGL canvas, providing a smooth and engaging visual effect.
- **Color Picker:** Users can change the ball's color using a built-in color picker with a range of color options.
- **Speed Adjustment:** The speed of the ball's movement can be adjusted using a slider.
- **Pause/Resume Functionality:** Users can pause and resume the ball's motion with a button.
- **Sound Effects:** A sound is played whenever the ball bounces off the edges. Users can toggle the sound on or off.
- **Shadow Effects:** The ball casts a shadow on a plane, enhancing the visual realism.
- **Responsive Design:** The UI panel is designed to be responsive and aesthetically pleasing.

## Getting Started

To get started with the Bouncing Ball EMDR Therapy Tool, follow these instructions:

### Prerequisites

- A modern web browser that supports WebGL.
- Ensure you have a sound file named `bounce.mp3` in the same directory as your `ball.js` file.


### Usage

1. **Change Ball Color:**
   - Use the color picker in the UI panel to select the desired color for the ball.

2. **Adjust Ball Speed:**
   - Use the slider to set the speed of the ball's movement.

3. **Pause/Resume Ball Motion:**
   - Click the "Pause" button to pause the ball's motion and click it again to resume.

4. **Toggle Sound:**
   - Click the "Sound On" button to toggle the bounce sound on or off.

### Files

- `index.html`: Main HTML file that includes the structure and UI components.
- `styles.css`: CSS file for styling the UI panel and other elements.
- `ball.js`: JavaScript file containing the WebGL setup and animation logic.
